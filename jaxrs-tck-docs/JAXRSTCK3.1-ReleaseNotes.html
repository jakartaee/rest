Jakarta EE RESTful Web Services Technology Compatibility Kit, Version 3.1
Release Notes, January 2022

Release Notes Update For Eclipse Jersey 3.1.0 Compatible Implementation Release January 2022

A couple of changes made in Eclipse Jersey 3.1.0 Compatible Implementation (CI) prompts this note.

  • The Eclipse Jersey 3.1.0 release requires the use of Java SE 11 and this is also required when running the Jakarta EE RESTful Web Services TCK against Jersey 3.1.

Kit Contents

The Jakarta EE RESTful Web Services 3.1 Technology Compatibility Kit (TCK) includes the following items:

  • Jakarta EE RESTful Web Services TCK, signature, API, and specification tests:

    • A signature test that checks that all of the public APIs are supported or defined in the Jakarta EE RESTful Web Services 3.1 implementation that is under test.

    • Specification tests for the testable assertions required by the Jakarta EE RESTful Web Services 3.1 Specification.

    • API tests for the public APIs and language elements defined by the Jakarta EE RESTful Web Services 3.1 Specification.

Jakarta EE RESTful Web Services 3.1 TCK Facts

The test suite bundle contains the following:

  • 2792 total tests

Platform Notes

The Jakarta EE RESTful Web Services 3.1 TCK tests and the Jakarta EE RESTful Web Services 3.1 Compatible Implementation (CI) can be run with Java Platform, Standard Edition version 11.0 (Java SE 11). See Compatible Runtime for Jakarta EE RESTful Web Services 3.1 for additional information.

Refer to the Technology Compatibility Kit User's Guide for Jakarta EE RESTful Web Services (HTML, PDF) for platform compatibility requirements and notes.

Compatible Runtime for Jakarta EE RESTful Web Services 3.1

A Compatible Runtime for compatibility testing of implementations based upon the Jakarta EE RESTful Web Services 3.1 Specification is as follows:

  • Java SE 11 which must be used with a Jakarta EE profile/application server, such as the GlassFish 6.1+ (with Java SE 11+)

  • A Compatible Implementation (CI) of Jakarta EE RESTful Web Services 3.1 is available as part of the Eclipse GlassFish 6.1+ full profile, or the Eclipse GlassFish 6.1+, Web Profile

The Jakarta EE RESTful Web Services 3.1 TCK tests have been run on the following platforms:

  • CentOS Linux 7
