Jakarta EE RESTful Web Services 3.1 TCK Documentation

The Jakarta EE RESTful Web Services 3.1 TCK documentation includes the following:

  • The Jakarta EE RESTful Web Services 3.1 Release Notes, provide late-breaking information about the Jakarta EE RESTful Web Services 3.1 TCK

  • The Jakarta EE RESTful Web Services 3.1 Technology Compatibility Kit User's Guide (or in PDF) provides the information that you need to install, set up, and run the Jakarta EE RESTful Web Services 3.1 TCK. The guide also provides the rules with which you must comply to pass the Jakarta EE RESTful Web Services 3.1 TCK.

  • The Javadoc Assertion List includes the javadoc assertions that are indirectly tested by the Jakarta EE RESTful Web Services 3.1 TCK.

  • The Specification Assertion List includes the specification assertions that are tested by the Jakarta EE RESTful Web Services 3.1 TCK.
